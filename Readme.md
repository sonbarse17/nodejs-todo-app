# Node.js Todo App

Todo application with EJS templates and CRUD operations.

## Features
- Express.js server
- EJS templating
- Todo CRUD operations
- XSS protection

## Quick Start
```bash
docker build -t nodejs-todo-app .
docker run -p 8000:8000 nodejs-todo-app
```

Visit: http://localhost:8000
